#link jQuery#

**LinkIt** is a simple jQuery plugin to attach link to elements

##version##
0.2.0

##Usage##
    $('span').linkIt({
               href: 'http://test.com',
               text: 'click me',
               target: '_self'
            })
   
##Vendors##
jQuery - [http://jquery.com](http://jquery.com)
